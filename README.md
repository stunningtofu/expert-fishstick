# Vue Tailwind Refactor

Refactor of a static portfolio website into **Vue 3 + Tailwind CSS**.

## Tech Stack

- Vue 3
- Vite
- Tailwind CSS
- pnpm

## Color Palette

From ColorHunt:

- #15173D
- #982598
- #E491C9
- #F1E9E9

## Install

Install dependencies using pnpm

```
pnpm install
```

Run development server

```
pnpm dev
```

Build production

```
pnpm build
```

Preview build

```
pnpm preview
```
